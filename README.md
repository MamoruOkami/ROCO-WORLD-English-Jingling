# 洛克王国宠物图鉴

一个基于 HTML + CSS + JavaScript 实现的洛克王国宠物图鉴工具，支持宠物浏览、属性筛选、宠物详情查看等功能。

## 功能介绍

- 宠物列表展示
- 属性筛选（火、水、草、电、翼、武、毒、土、萌、龙、恶魔、机械等）
- 宠物详情页（种族值、特性、技能、威力、消耗）
- 响应式布局，适配电脑与手机端
- 技能图标、特性图标、属性图标自动匹配
